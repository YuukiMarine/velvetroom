## 这是什么？

一款以JRPG游戏为精神内核的**本地优先个人成长追踪器**。

它将 RPG 角色成长机制移植到真实生活：记录你做的事 → 转化为属性点数 → 积累升级 → 解锁技能与成就 → 用 AI 回顾你的成长弧线。

## 核心系统

### 人格五维属性

| 属性 | 默认名 | 代表的行为 |
|------|--------|-----------|
| Knowledge | 知识 | 阅读、学习、思考 |
| Guts | 胆量 | 挑战、运动、意志力 |
| Dexterity | 灵巧 | 技艺、创作、动手能力 |
| Kindness | 温柔 | 关怀、帮助、共情 |
| Charm | 魅力 | 社交、表达、气场 |

五项属性**全部可自定义命名**，初始化引导提供预设风格，也可完全自定义。  
属性每升一级可解锁对应技能，技能激活后为该属性的加点提供**额外百分比加成**。

---

### 记录事项（历史记录）

- 用自然语言描述你的行动，**关键词规则**自动识别并建议加点分配
- 手动精细调整每项属性分值（0–5），一次记录可同时影响多项属性
- 标记**重要事件**（⭐），在日历热力图中以琥珀色圆点高亮显示
- **日历热力图**直观呈现活跃程度，色点大小随当日总点数变化，支持年/月跳转与当日记录展开
- 历史记录按年 → 月 → 日三级折叠归档，支持全文搜索、多维筛选（属性 / 来源 / 时间 / 重要程度）
- 滚动到列表底部时触发**波浪反馈动画**，随后自动展开下一个折叠的月份
- 每条记录**长按可删除**，带二次确认保护

---

### 待办系统

- 每个待办绑定属性与奖励点数，完成即自动加点并写入历史
- 四种模式：
  - **单次完成**：完成即归档
  - **每日重复**：每日重置，永不归档
  - **多次计数**：当日累计完成 N 次后触发加点
  - **长期目标**：跨日累计，达到总次数后触发加点
- 支持**按星期过滤**：指定待办仅在特定星期几出现
- **重要待办**（⭐）置顶显示，卡片以琥珀色高亮
- 长按待办卡片 500 ms 进入编辑模式
- 已归档待办可恢复为活跃状态

---

### 本周目标

- 每周可设定一组目标（至少选择 2 项，最多 4 项），目标类型包括：
  - **活动次数**：指定属性的记录次数达到 N
  - **待办完成**：本周完成待办 N 次
  - **属性点数**：指定属性获得 N 点
  - **全属性点数**：所有属性总获得点数达到 N
- 每项目标独立显示进度条与当前值/目标值
- 目标卡片**实时更新进度**，全部达成后出现"领取奖励"按钮
- **完成庆祝弹窗**：彩色粒子爆发 + 选择奖励属性（Lv.3+ 属性获得 +7，其余 +5）
- **领取确认动画**：扩散光环 + 浮现属性名与加点数，2.6 秒后自动关闭
- 可设置**自定义奖励**
- 长按目标卡片弹出操作菜单：修改选项 / 重置 / 删除（完成后仅可删除）
- 过期未完成的目标在下次打开应用时**自动清除**

---

### 成就 & 技能

**成就**在达成条件后需要**手动点击解锁**——给自己一个值得记住的仪式感瞬间。

内置成就条件类型：

| 类型 | 说明 |
|------|------|
| `consecutive_days` | 连续打卡天数（最长连续记录） |
| `total_points` | 累计总点数 |
| `attribute_level` | 指定属性达到目标等级 |
| `keyword_match` | 活动描述包含指定关键词 |
| `todo_completions` | 待办完成总次数 |
| `weekly_goal_completions` | 完成每周目标的次数 |
| `all_attributes_max` | 所有属性同时达到最高等级 |

内置成就（15 项）：连续打卡 7 天 · 完成待办 10 次 · 累计 100 点 · 五项属性各自 Lv.3 / Lv.5 · 全属性满级 · 完成 8 次每周目标（📅 计划通）

支持**自由创建、编辑自定义成就和技能**，内置成就的标题、图标、描述也可编辑。

**技能**：每项属性有两个内置技能（Lv.3 解锁 +20% 加成，Lv.5 解锁再叠加 +10%），多个技能效果**乘算叠加**。

**馆长的赐福**（初始化时选择的专属技能）：为指定属性每次加点额外 +1，可在技能页随时开启/关闭。

### AI 成长总结

> *"让我来检视你的记录，客人……"*

每逢**周末、月初、年末**，历史记录页右上角将亮起红点提示，召唤你进行周期性总结。

| 特性 | 说明 |
|------|------|
| **流式逐字输出** | 字符实时流入，无需等待整段生成 |
| **一次追问机会** | 总结完成后可向 AI 追问一个问题 |
| **年度总结** | 12 月 31 日偷偷出现专属年度总结入口 |
| **归档保存** | 每份总结可存入档案，随时重读或删除 |
| **多 API 支持** | 兼容 OpenAI / DeepSeek / Kimi 及任意 OpenAI 格式 API |

**内置两种风格预设：**

- 🎭 **馆长伊戈尔**：庄严睿智，将你的每次成长比作"灵魂的觉醒"，偶有神秘感
- 📋 **成长助手**：数据导向，关注属性均衡，给出 3–5 条具体可执行的行动建议

支持编辑内置预设，亦可添加完全自定义的 System Prompt 风格。

## 其他亮点

### 首页仪表盘

| 功能 | 说明 |
|------|------|
| **时段问候语** | 按时段（晨/午/昏/夜等 7 个时段）随机显示不同问候与副文案 |
| **今日待办进度** | 问候条内嵌完成进度（N/M 条），点击待办直接完成并加点 |
| **每日事件** | 每天随机一个加成事件，为某属性当日加点乘以倍率（最高 ×2） |
| **逆流预警** | 逆流功能开启时，属性连续 3 日无增长则在首页提前一天显示预警提示 |
| **逆流衰减通知** | 衰减触发后弹窗展示受影响属性与扣减量 |
| **属性卡片** | 支持拖拽排序 + 全宽展开，顺序持久保存 |
| **成长概览** | 五边形雷达图 + 六项统计数据（总点数、连续天数、成就数、技能数等），可跳转统计页 |

### 设置系统

| 功能 | 说明 |
|------|------|
| **主题系统** | 5 种主题色（蓝/黄/红/粉/自定义）一键切换 |
| **自定义主题色** | HEX 色值输入 + 色盘选取，实时预览 |
| **自定义音效方案** | 自定义主题下可独立选择音效风格（P3蓝/P4黄/P5红/P3粉） |
| **逆流（实验性）** | 连续 3 日无增长属性次日自动 −1，开启后有 3 日缓冲期 |
| **背景动画（多选）** | 极光 / 粒子 / 渐变波 / 脉冲，可同时叠加，随主题色变化 |
| **深色模式** | 全界面深色适配 |
| **背景图片** | 上传本地图片，可调透明度、横竖屏模式，备份时自动剥离 |
| **关键词规则** | 每个属性维护一组关键词，记录时自动分析并建议加点 |
| **升级阈值编辑** | 可修改每级所需点数，最多扩展至 Lv.10 |
| **数据导入/导出** | 一键复制 JSON / 下载备份 / 粘贴恢复 / 文件拖拽导入，含原子回滚保护 |

### 统计页（成长概览）

- **成长轨迹面积图**：以日为单位绘制累计总点数曲线
- **属性趋势多线图**：可选 7 日 / 30 日 / 全部，支持按属性单独开关
- **属性分布横向条形图**：按总点数排序，显示等级与占比
- 四项汇总数据卡片：累计点数 / 记录天数 / 最长连续打卡 / 日均点数

## AI 总结配置

在「**设置 → AI 总结**」中配置：

1. **API 提供商** — 选择 OpenAI / DeepSeek / Kimi，或填写自定义 Base URL
2. **API 密钥** — 仅存储于本地 IndexedDB，不上传任何服务器
3. **模型名称**（可选，留空使用各平台默认模型）
4. **风格预设** — 选择或编辑 System Prompt

> **修改内置预设的 System Prompt：**  
> 方式一：`设置 → AI 总结 → 风格预设 → 编辑`（推荐）  
> 方式二：直接编辑 `src/store/index.ts` 中的 `DEFAULT_SUMMARY_PROMPT_PRESETS`

---

## 数据与隐私

- 所有记录、属性、成就数据均存储于**本机 IndexedDB**，不依赖任何云服务
- 导出备份时自动剥离背景图（base64）与 API Key 等敏感字段
- 导入备份具有**快照回滚保护**：若写入过程失败，自动完整恢复原有数据
- 导出后的 JSON 文件可在不同设备间迁移
