# 快速开始指南

## Windows 用户

双击运行 `start.bat` 文件，它会自动：
1. 检查 Node.js 环境
2. 安装所有依赖
3. 启动开发服务器

## Mac/Linux 用户

在终端中运行：
```bash
chmod +x start.sh
./start.sh
```

## 手动启动

如果自动脚本无法运行，请手动执行：

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev
```

## 访问应用

浏览器访问: `http://localhost:5173`

## 构建生产版本

```bash
npm run build
npm run preview
```
